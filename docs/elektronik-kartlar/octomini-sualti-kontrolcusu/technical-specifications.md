# Ürünün Teknik Özellikleri

## Kart Hakkında

| Özellik              | Değer                 |
|----------------------|-----------------------|
| Çıkış pwm pin sayısı | 5+1 Pin               |
| Mikro denetleyici    | Arduino Nano          |
| Dengeleme            | Gyro - Mpu6050        |
| İletişim modülü      | Canbus                |
| Mesafe               | 400 metre (teorik)    |
| Besleme voltajı      | 7-12 Volt             |
